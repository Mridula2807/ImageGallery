# 🐾 Pets Gallery Website

An attractive and user-friendly multi-page Pets Gallery website developed using HTML and CSS. This project showcases different pets with image-based navigation, clean layouts, and modern styling techniques.

## ✨ Features

* Beautiful pets image gallery
* Multi-page website navigation
* Clean and responsive layout
* Social media icons integration
* Organized folder structure
* Simple and interactive user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons

## 📂 Project Structure

* `index.html`
* `pages/`
* `assets/css/`
* `assets/img/`

## 🎯 Project Objective

The purpose of this project is to practice front-end web development concepts such as webpage structure, styling, navigation, image handling, and UI design.
